Day2

目前所有组件所实现的功能

  1. App.vue (根组件)
  • 技术点：
    • Vue 3 组件
    • Vue Router
  • 功能：
    • 作为应用的根组件
    • <router-view/> 用于渲染路由匹配的组件
    • 是整个应用的入口组件
  1. src/layout/index.vue (布局组件)
  • 技术点：
    • Element Plus 组件库
    • SCSS 样式预处理器
    • Vue 3 组件
  • 功能：
    • 实现后台管理系统的整体布局
    • 使用 Element Plus 的布局组件：
      • el-container: 整体容器
      • el-aside: 侧边栏（宽度 200px）
      • el-header: 顶部导航
      • el-main: 主内容区域
    • 使用 SCSS 嵌套语法定义样式
  1. src/router/index.ts (路由配置)
  • 技术点：
    • Vue Router
    • TypeScript
    • 路由类型定义
  • 功能：
    • 配置应用的路由系统
    • 使用 createWebHistory 创建 HTML5 历史模式
    • 定义路由规则和嵌套路由
    • 实现页面导航和组件切换
  1. src/components/HelloWorld.vue (示例组件)
  • 技术点：
    • Vue 3 组件
    • Composition API
    • Pinia 状态管理
    • Element Plus UI 组件
  • 功能：
    • 展示计数器功能
    • 使用 Pinia 管理状态
    • 实现数据修改的多种方式：
      • 直接修改 store
      • 使用 actions
      • 使用 $patch
      • 使用 $patch 函数式更新
  1. src/main.ts (应用入口)
  • 技术点：
    • Vue 3
    • Pinia
    • Element Plus
    • TypeScript
  • 功能：
    • 创建 Vue 应用实例
    • 注册全局插件和组件
    • 配置 Pinia 状态管理
    • 挂载应用到 DOM

整体技术栈：

  1. 前端框架：Vue 3
  2. 开发语言：TypeScript
  3. UI 框架：Element Plus
  4. 状态管理：Pinia
  5. 路由管理：Vue Router
  6. 样式处理：SCSS
  7. 构建工具：Vite

这是一个典型的现代化 Vue 3 后台管理系统的基础架构，采用了当前流行的技术栈，具有以下特点：

  • 使用 TypeScript 提供类型安全
  • 使用 Composition API 提供更好的代码组织
  • 使用 Pinia 进行状态管理，替代了 Vuex
  • 使用 Element Plus 提供美观的 UI 组件
  • 使用 SCSS 提供更强大的样式处理能力
