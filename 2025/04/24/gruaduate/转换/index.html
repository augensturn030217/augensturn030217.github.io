用户权限相关表

用户表（sys_user）

字段名称 数据类型 是否主键 是否为空 字段描述
user_id int(11) 是 否 用户ID
username varchar(36) 否 是 登录账户
password varchar(128) 否 是 登录密码
phone varchar(18) 否 是 电话
email varchar(36) 否 是 邮箱
sex varchar(4) 否 是 性别
is_admin varchar(2) 否 是 是否是超级管理员
is_account_non_expired tinyint(4) 否 是 账户是否过期
is_account_non_locked tinyint(4) 否 是 账户是否被锁定
is_credentials_non_expired tinyint(4) 否 是 密码是否过期
is_enabled tinyint(4) 否 是 账户是否可用
nick_name varchar(36) 否 是 姓名
create_time datetime 否 是 创建时间
update_time datetime 否 是 更新时间

字段名称 数据类型 是否主键 是否为空 字段描述

url varchar(128) 否 是 组件路径
type varchar(2) 否 是 菜单类型
icon varchar(36) 否 是 菜单图标
parent_name varchar(36) 否 是 上级菜单名称
order_num int(11) 否 是 序号
create_time datetime 否 是 创建时间
update_time datetime 否 是 更新时间
